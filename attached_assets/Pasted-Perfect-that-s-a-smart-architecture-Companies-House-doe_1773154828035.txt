### 2. Source Priority for Office Mode

**Step 1 — Companies House API (primary discovery)**
- Search by SIC code and registered address/postcode for the target geography
- Extract: company name, company number, registered address, active directors (contact name), incorporation date
- Filter to active companies only
- Use SIC codes covering: consulting, technology, software, legal, accounting, recruitment, marketing, engineering, financial services, architecture, planning, HR, PR, environmental, compliance, insurance, corporate finance

**Step 2 — Google Places cross-reference (website lookup only)**
- For each company returned by Companies House, query Places API with `company name + town`
- If a match is found with confidence, extract: website URL, phone number, Google rating
- Do NOT use Places as a discovery source — only to enrich existing Companies House records
- This keeps Places API calls to a minimum (one lookup per CH record, not one per sector per town)

**Step 3 — Website scraping (enrichment)**
- Once a website URL is identified, scrape for:
  - Named contacts (About/Team/Contact pages)
  - Generic email (`info@`, `hello@`, `contact@` etc.)
  - Generate `personal_email_guesses` (pipe-separated, all common formats)
- Companies House director name takes priority over website-scraped names if both found
